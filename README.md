# Excel Data Analysis Project

## About the Project
This project is developed as part of my internship program. It focuses on analyzing Excel data to extract useful insights and support decision making.

## Objective
- Clean and organize raw data
- Perform data analysis using Excel
- Identify trends and patterns
- Create charts and pivot tables

## Tools Used
- Microsoft Excel
- Excel formulas (SUM, IF, VLOOKUP, AVERAGE)
- Pivot Tables
- Charts

## Dataset Information
- Internship provided dataset
- Excel (.xlsx) file
- Structured business data

## Steps Performed
1. Data cleaning (duplicates, missing values)
2. Data formatting
3. Excel formulas applied
4. Pivot tables created
5. Charts made
6. Insights generated

## Key Insights
- Top performing category identified
- Trends observed in data
- High and low performance areas found
